# Flask App Boilerplate Code

## Overview
Simple Flask boilerplate code as a quick-start to writing new apps.

## Dependencies
- Linux or macOS
- Python 2.7
- Virtualenv

## Clone
```git clone https://github.com/shrir/flask-boilerplate.git```

## Install
```make install```

## Test
```make test```

## Run
```make start```

## Stop
```make stop```

## Uninstall
```make clean```
